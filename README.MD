# TAREA 1

Mi primera página web semántica y estilizada

## Descripcion

Este proyecto es una tarea de nivelacion del curso de react js, en el cual se nos pide como objetivo lo siguiente:

- Crear una página HTML con estructura semántica correcta y estilos básicos en CSS, aplicando etiquetas, clases, selectores, variables y buenas prácticas.

## Estructura

- index.html: Contiene la estructura de la página.
- style.css: Contiene los estilos de la página.
- app.js: Feat future

## Clonar Repo y Configurar el Entorno

- Dentro de la consola de powershell en visual studio code u el IDE de confianza que utilices ejecutamos el comando , gh repo clone Bruno395dev/tarea_1
- Una vez clonado el repositorio debe instalar la extension live server de ritwickdey en su IDE.
## Ejecutar

Opcion 1: en Visual Studio Code 
- Clic derecho sobre index.html -> Open with Live Server

Opcion 2: en Windows
- Doble click al archivo index.html en el explorador de windows
